### 문제 설명

<p>You are required to find a palindome string with a minimal length which matches a given template for search. Palindrome is a string which can be read in the same way in both directions (forward and backward). An empty string is also a palindrome. The template can contain lower case latin letters corresponding to the same letters in a string, symbol '<code>?</code>' corresponding to an arbitrary latin letter and symbol '<code>*</code>' corresponding to a zero or more arbitrary latin letters.</p>
